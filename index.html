Demo Educativa de Permisos Web

NOTA: Esta es una aplicación educativa que demuestra cómo los sitios web pueden solicitar permisos de cámara. Utilizar este conocimiento para engañar o espiar a otros es ilegal y no ético.
Introduce una URL válida de YouTube
Cómo funciona:
  1. Ingresa una URL de YouTube
  2. Se generará una página con ese video
  3. Cuando alguien visite la página, se le pedirá permiso para acceder a su cámara
  4. La imagen capturada se mostrará en tu pantalla
